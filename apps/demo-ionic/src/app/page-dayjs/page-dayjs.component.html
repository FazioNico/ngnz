Immutable Date-Time utils allow you to display Date like 1 minute ago.

Simple demo

 Using `fromNow`

Time form 01/01/2021 to now

{{'01/01/2021'|dayjs: 'fromNow'}}

Get time from X to Y

 Using `to`

Time from 01/01/2021 to 01/04/2021

{{'01/01/2021'|dayjs: 'to': '01/04/2021'}}

X is after Y

 Using `isAfter`

01/01/2021 is after 01/04/2021

{{'01/01/2021'|dayjs: 'isAfter': '01/04/2021'}}

Date formating

 Using `format`

01/01/2021 formated with format YYYY

{{'01/01/2021'|dayjs: 'format': 'YYYY'}}
